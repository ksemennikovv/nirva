# Компонент: Плеер медитаций (Med Player)

## Что это

HTML5 аудиоплеер для воспроизведения медитаций. Переиспользуется на странице медитаций и в карточках. Умеет отслеживать прогресс прослушивания и отправлять статистику на сервер.

---

## Ключевые файлы

```
features/med-player/
├── index.php (или встраиваемый фрагмент)  # HTML-разметка плеера
├── med-player.css                          # Стили плеера
└── med-player.js                           # Логика воспроизведения
```

---

## Что умеет

- Play / Pause
- Перемотка (seekbar)
- Отображение текущего времени и длительности
- Автостоп в конце
- Отслеживание прогресса — сколько секунд прослушано
- При завершении (или при остановке после 80%+ прослушивания) — отправляет статистику

---

## Как встраивать

```php
$meditation = $meditationRepository->getById($id);
require_once 'features/med-player/index.php';
```

Компонент читает `$meditation['audio_url']` и `$meditation['id']`.

---

## Статистика прослушиваний

По завершении воспроизведения JS отправляет POST:
```
POST /api/meditations/record-listen.php
{
  meditation_id: 123,
  duration: 245,      // секунд
  completed: true     // дослушал до конца?
}
```

Сервер вызывает `MeditationRepository::recordListen()` — запись в `meditation_listens`.

Данные используются для:
- Статистики пользователя на дашборде
- Аналитики в админке
- `getLastListenedPerMeditation()` — показывает когда последний раз слушал
